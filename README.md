Developed a full-stack web application, YelpCamp, that allows users to create, view, and review camping sites. Implemented user authentication and authorization using Passport.js and secure session handling with Express. Integrated Mapbox API to enable location tagging and map visualization for each campground. Designed RESTful routes and implemented CRUD operations with MongoDB for data persistence. Focused on modular backend architecture, form validation, and server-side rendering using EJS templates to ensure a seamless user experience.
